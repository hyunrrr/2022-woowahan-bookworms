## 다양한 예제로 배우는 CSS설계 실전 가이드 76-86

### 3장 다양한 CSS 설계

### 1. OOCSS (Object Oriented CSS)
- 레고처럼 자유로운 조합이 가능한 모듈의 집합을 만든다.
- 그 모듈을 조합해 페이지를 만든다.
- 따라서, 신규페이지를 만드는 경우에도 기본적으로 추가로 CSS를 만들 필요가 없다.


#### 스트럭쳐(구조)와 스킨(화면) 분리
- 스트럭쳐에 해당하는 속성
    - width
    - height
    - padding
    - margin
- 스킨에 해당하는 속성
    - color
    - font
    - background
    - box-shadow
    - text-shadow

#### 컨테이너와 콘텐츠 분리
- 컨텐츠는 대략 '영역', 콘텐츠는 '버튼'모듈 
- 모듈을 가능한 특정한 영역에 의존하지 않도록 한다.
